Material and Resources

The course is based on various Coursera courses. It may be seen as a subset of the DeepLearning Specialization on Coursera, available also at DeepLearning.ai (it may slightly differs). However, both specialization has a lot of contents and are not for free (even though on Coursera the course can be audit for free).
In this course we will go through the first 3 Courses:
  • Neural Networks and Deep Learning,
  • Improving Deep Neural Networks: Hyperparameter tuning, Regularization and Optimization,
  • Structuring Machine Learning Projects

The course will make use of the Google Colab tool for the programming part, which allow for a very smooth collaborative working environment with no need to install anything on the personal computer and access to GPU computing.
The programming will be done in the Python language. Everybody should have basic knowledge in this language (or Matlab, R and similar) in order to be able to complete the assignment. Working with array and matrices is needed and is very well handled from the numpy python library.
In the second week of the Neural Networks and Deep Learning course a further insight on vectorization in python is given, so no need to worries if you are not yet very familiar with such operations. Some of the concepts explained in the course will require some basic knowledge of mathematics and linear algebra. Again, the first 2 weeks of the Neural Networks and Deep Learning course will give hopefully enough examples to clarify the matter. On request, additional bibliography can also be provided.

Useful tools:

  • Github and Gitlab: online hosting for repositories with version control using git (this may be annoying at the beginning, but then it's great!).
    They are very similar, with small differences that may be subjective. Lately gitlab has been growing a lot and it may offer more options for deployment and testing of the code.
    I give here references to github, yet the same things can be made also with gitlab, the choice is yours.
    You can use it to create a repository upload your code, keeping it either private or public, collaborate with others, create static webpages for your projects (like this one), create wikis for your software, and much more.
  • Google Colab: collaborative coding environment with access to GPU computing power. We will use this during the course - it is strictly related to the coursera courses and provided there as a tool to complete the assignment. It uses Python Notebooks to allow the creation of an interactive document where code and text can be combined.
  • Python Notebooks can also be used offline with Project Jupyter, installing it on your own computer, and notebooks can be uploaded to github and they will be rendered as static HTML. That means that images and plot you made will be also visibles from others.
    If you want it to be interactive, Binder is an awesome tool that allows to do that very quickly, and support also virtual environments.

FAQ

How can I register to the Mattermost Channel for the Course?

The registration is unfortunately separate from the OpenCampus registration, and can be made here: Link for registration.
You can create a different user account. Once you are registered, please join the General Channel for the Summer Semester and the Deep Learning Channel.

Where is the course?

The course is online and will be given via the Zoom platform every Monday at 16.00. The lectures are open, and the link is the following: Link to Join the Zoom Meeting.

Can I follow the course also if I am not online on Mondays?

In theory yes, but the best parts are the discussion, who will happen online on Mondays. Of course you can follow the online Coursera courses and then ask question on the Mattermost channel and still participate, if you like it this way.

Will the course be recorded?

No, at the moment this is not planned. The slides will be uploaded in this page and some external links to material and resources may also be available here, but the discussions will not be fully available.
